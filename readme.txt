Turn Images into coloring pages

== Description ==

Pixobe coloring book, turn any images into coloring book.

== Features ==
 1. Paint
 2. Draw
 3. Print
 4. Download
 5. Zoom
 6. Undo
 7. Color transparency
 8. Pencil Thickness

 == Change Log
1. Users can upload their own images, pass upload=true to the shortcode.
2. Coloring app now supports color picker, pass color-picker=true in the shortcode to enable color picker.
